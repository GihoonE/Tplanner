# 개발 기록 · 2025-03-22

## DB · Prisma

- Prisma + SQLite 연결 (`prisma/dev.db`)
- `Student`, `Session`, `HomeworkItem` 스키마 추가
- 시드: `npm run db:seed` — 학생 5명, 수업 8건

## API

| 경로                        | 메서드             | 설명                                                                                                                     |
| --------------------------- | ------------------ | ------------------------------------------------------------------------------------------------------------------------ |
| `/api/students`             | GET, POST          | 학생 목록/추가 (lastSessionAt, thisMonthSessionCount 포함)                                                               |
| `/api/sessions`             | GET, POST          | 수업 목록(`?studentId=1` 필터) / 새 수업 생성(body: studentId, start, end, place, notes, understanding, focus, homework) |
| `/api/sessions/[sessionId]` | GET, PATCH, DELETE | 단건 조회/수정/삭제                                                                                                      |

## 페이지 · 컴포넌트

- **학생 관리** (`/students`): store → API 데이터 사용
- **SessionModal**: store → API 단건 조회 + PATCH/DELETE
- **Records** (`/records`): RecordList에 검색 추가 (이름, 과목, 메모)

## 유의사항

- `params`는 Next.js 14 기준 `Promise` 아님 (15+부터 Promise)
- URL/쿼리 값은 항상 `string` → 숫자 필요 시 `parseInt(id, 10)`
